## sass
### mixin
- why

复用css

- 定义
```sass
@mixin button-size($padding-x, $padding-y, $font-size, $border-radius) {
	border-radius: $border-radius;
	padding: $padding-y $padding-x;
	font-size: $font-size;
}
```
- 使用

```sass
@include button-size(
		$btn-padding-y,
		$btn-padding-x,
		$btn-font-size,
		$border-radius
	);
```
